# xss-vuln
xss-vuln is a simple reflected xss scanner
# Install 
```
git clone https://github.com/htw-lamer111/xss-vuln
cd xss-vuln 
pip3 install -r requirements.txt 
python3 main.py 
cd web
python3 -m http.server
```
